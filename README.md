## Como usar:

* Importe o schema MYSQL na pasta raiz.

* Para rodar a aplicação execute no terminal:
``` 
$ php -S localhost:8080 -t /mvc-crud/web/
```

* Acesse no seu browser `http://localhost:8080/`
